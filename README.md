# Unity Design Patterns

## Description

A set of reusable architectural patterns for Unity Projects.

For now contains:

- Bloc (Business Logic Component) : port of original **dart/flutter** [bloc](https://bloclibrary.dev/)
- State Machine
- Service Locator
- Injector
- Reactive Property 
- Heap data structure
